Who we are

We're a group for all things Python, open-source and STEM.

Whether you've never programmed, or have been programming for a long time - please join us!

We're always looking for people for presentations, so if you have something you'd like to talk about, please let us know!

Tell us in advance, and we can announce the topic beforehand, otherwise we always have an open floor for discussion.

There really isn't a high bar to participate; this is great space to bring up things like:

  • projects (we like to hear about works-in-progress too)
  • code snippets & language feature examples
  • a problem you're stumped by
  • an interesting library

Our first talk

Coding for everyone: a practical deep-dive into the modern web page

Discover the coding power-tool hidden in your computer's web browser. A demo loaded with practical tips and critical concepts.
See how to tweak web pages. Learn about software design. Find out if the coding craze is for you--or just see what all the fuss is about!
We present very basic HTML, CSS, and JavaScript, leveraging a built-in browser tool to modify a webpage.
For high school through retirement.

Don't want to learn to code but are curious? Might want to learn but aren't sure?
Wondering what coding looks like, or what kind of person does it?
Thinking that some understanding is important for citizens in the digital age?
Then check out this talk!
